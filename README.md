# Lab3_Recipe_Explorer_Application

This project is a simple Android application built with Jetpack Compose that allows users to explore recipes
and view their details.

Features:
- Displays a scrollable list of recipes using LazyColumn
- Each recipe is shown inside a material3 card
- Clicking a recipe opens the recipe detail screen
- Navigation compose is used for screen navigation
- ViewModel and StateFlow are used for state management
- Adaptive UI:
- Phone: List screen > Detail screen navigation
- Tablet: Split screen layout with list on the left and details on the right side.

Technologies:
- Kotlin
- Jetpack Compose
- Material3
- Navigation Compose
- ViewModel
- StateFlow

How to Run:
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle
4. Run the app on an emulator or Android device.
